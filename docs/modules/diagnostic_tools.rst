Diagnostic and Analysis Tools
==============================

Prescription Analysis
---------------------

Extract and analyze information from prescription images.

Features
~~~~~~~~

- Extract medicines from prescription images
- Parse dosage and frequency information
- Generate detailed medicine information
- Batch process multiple medicines

Medical Tests Graph
-------------------

Build and visualize knowledge graphs of medical tests and their relationships.

Features
~~~~~~~~

- Extract test relationships using AI
- Build NetworkX knowledge graphs
- Visualize relationships
- Query test relationships

Physical Exam Questions
-----------------------

Access clinical questions for physical examinations.

Features
~~~~~~~~

- Standardized physical exam questions
- Evidence-based clinical assessment
- Documentation support
